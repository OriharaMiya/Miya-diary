📝 原始记录 · Raw Log

🧠 深度心理分析 · In-Depth Analysis
1️⃣ **认知-情绪失调** 理性知道“该上班”，情绪却完全失动力 → 典型 Cognitive Dissonance。 Rational mind says “work”; emotions refuse → classic cognitive dissonance. 2️⃣ **功能性抑郁 & 快感缺失** 能行动却感受不到愉快；多巴胺通路长期疲乏。 Functional-depression style: active body, numb reward system. 3️⃣ **羞耻-耽搁循环** 拖延→自责→更焦虑→继续拖延。源自童年“做不好就被批评”的内在脚本。 Procrastination → guilt → more anxiety → more delay, rooted in early “criticism if not perfect”.
📌 建议 · Advice
▪ 把任务拆成 10 分钟微目标，降低启动门槛。 ▪ 每日写一句“我允许自己休息”的肯定句；破除羞耻。 ▪ 若再次陷入拖延→自责，立即做 3 次深呼吸 + 写下当前感受。 Do 10-min micro-tasks; daily self-permission mantra; when procrastination loop starts, 3 deep breaths + jot feelings.
